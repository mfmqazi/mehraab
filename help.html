ü§ñ Using the AI Content Generator

The AI Content Generator is your most powerful tool for creating study materials.

Step-by-Step Guide:

Step 1: Navigate to the AI Generator

Click "AI Generator" in the top menu or the "Generate Notes" button.

Step 2: Select Your Subject

Choose from Medicine, Surgery, Pediatrics, ObGyn, ENT, or Ophthalmology.

Step 3: Choose a Chapter

Select the chapter you want to study from the dropdown menu.

Step 4: Specify a Topic (Optional)

Enter a specific topic like "Myocardial Infarction" for focused content.

Step 5: Configure Options

Set number of practice questions, include mnemonics, and clinical pearls.

Step 6: Generate Content

Click "üöÄ Generate Content" and wait 5-10 seconds for AI to create your notes.

Step 7: Download or Copy

Save the generated content as JSON or copy it to your clipboard.

What You'll Get:

  • Key Points: 7-10 high-yield points covering the topic comprehensively
  • Mnemonics: Memory aids to help you remember complex concepts
  • Clinical Pearls: Important clinical tips and exam-relevant information
  • Practice Questions: MCQs with detailed explanations
üí° Bulk Generation: Use the "üì¶ Bulk Generate" button to create content for an entire subject at once. This is great for comprehensive preparation!

üì• Using Your Generated JSON Content

After generating content with AI, you have several ways to use it:

Method 1: Copy & Study Directly

Quick Review:

Click "üìã Copy JSON" and paste into a text editor or note-taking app for immediate study.

Method 2: Download & Save

Build Your Library:

Click "üíæ Download" to save the JSON file. Organize these files by subject/topic for future reference.

Method 3: Import into CMS

  1. Download the JSON file from AI Generator
  2. Navigate to CMS (Content Management System)
  3. Click on "Import/Export" in the sidebar
  4. Click "Choose File" and select your downloaded JSON
  5. Click "Import Content"
  6. Your content is now in the CMS where you can edit and organize it!

Method 4: Share with Classmates

üí° Collaboration:

Download JSON files and share them with your study group via WhatsApp, email, or cloud storage. Everyone can import the same content!

What's Inside the JSON?

Each generated JSON file contains structured data including:

  • Subject & Author: Reference textbook information
  • Chapter & Topic: Organized by medical specialty
  • Key Points: Array of high-yield facts
  • Mnemonics: Memory aids with explanations
  • Clinical Pearls: Important clinical tips
  • Practice Questions: MCQs with options, correct answers, and explanations
  • Metadata: Generation timestamp for tracking

Example JSON Structure:

{
  "subject": "Medicine",
  "author": "Archith Boloor",
  "chapter": "Cardiology",
  "topic": "Myocardial Infarction",
  "keyPoints": [
    "Definition and pathophysiology...",
    "Clinical presentation...",
    "Diagnostic criteria..."
  ],
  "mnemonics": {
    "MONA": "Morphine, Oxygen, Nitroglycerin, Aspirin"
  },
  "clinicalPearls": [
    "Always check for contraindications..."
  ],
  "practiceQuestions": [
    {
      "question": "A 55-year-old male presents with...",
      "options": ["A", "B", "C", "D"],
      "correctAnswer": 0,
      "explanation": "Detailed explanation..."
    }
  ]
}

üéì Tips for Exam Success

Study Strategy:

  1. Start Early: Begin preparation at least 3 months before exams
  2. Use AI Generator: Create comprehensive notes for all topics
  3. Practice Daily: Solve at least 20-30 MCQs every day
  4. Review Regularly: Use flashcards for quick revision
  5. Take Mock Tests: Simulate exam conditions weekly
  6. Track Progress: Monitor your performance and adjust strategy

Best Practices:

  • Focus on high-yield topics first
  • Understand concepts, don't just memorize
  • Use mnemonics for complex topics
  • Practice clinical scenarios regularly
  • Review your mistakes and weak areas
  • Take breaks to avoid burnout
üí° Study Schedule: Use the Pomodoro timer for focused 25-minute study sessions. Take 5-minute breaks between sessions and a longer 15-30 minute break after 4 sessions.

üîß Troubleshooting

  • Try refreshing the page

  • Don't use incognito/private mode
  • Check browser storage settings

Page loading slowly?

  • Clear browser cache
  • Close unnecessary tabs
  • Try a different browser
